Minecradvance

The ultimate, private Minecraft progress tracker. Fully open-source and built for the community.

About the Project

Minecraft Progress Tracker was created because players trying to complete "Adventuring Time" or "Monsters Hunted" lacked an elegant, fast, and secure tool. This app was engineered entirely in Angular 21 using Signals and standalone components to guarantee 100% client-side privacy.
