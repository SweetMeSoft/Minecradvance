# Minecraft Achievement Icons

This directory should contain the official Minecraft advancement icons extracted from the game's `.jar` file.

## How to Extract Icons

1. Locate your Minecraft installation:
   - **Windows:** `%appdata%\.minecraft\versions\<version>\<version>.jar`
   - **macOS:** `~/Library/Application Support/minecraft/versions/<version>/<version>.jar`
   - **Linux:** `~/.minecraft/versions/<version>/<version>.jar`

2. Open the `.jar` file with any ZIP extractor (7-Zip, WinRAR, etc.)

3. Navigate to: `assets/minecraft/textures/item/` and `assets/minecraft/textures/block/`

4. Copy the relevant icons listed below into this directory.

## Alternative: Download from Minecraft Wiki

You can also download icons from the [Minecraft Wiki](https://minecraft.wiki/) which provides high-resolution versions of all item and block textures.

## Icon Specifications

- **Format:** PNG with transparency
- **Size:** 64×64px recommended (upscaled from 16×16 game textures)
- **Naming:** Use the Minecraft resource ID as filename (e.g., `diamond.png`, `ender_pearl.png`)
